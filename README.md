# Loan Prediction ML App

## 📌 Problem Statement
This project predicts whether a loan will be approved based on applicant details.

## 🚀 Features
- User input form
- Real-time prediction
- Clean UI using Streamlit

## 🛠 Tech Stack
Python, Pandas, Scikit-learn, Streamlit,matplotlib, SHAP

## 📊 Model Used
Random Forest

## ▶️ How to Run
streamlit run loan.py
